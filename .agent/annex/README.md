## Purpose

The Skills system describes the **present state**. The Annex captures **why** that state emerged.

## Categories

### Decision Logs
Records of significant architectural or design decisions with rationale.
*   Format: `YYYY-MM-DD_decision_title.md`

### Post-Mortems
Analysis of incidents, bugs, or failed approaches.
*   Format: `YYYY-MM-DD_postmortem_title.md`

### Version Notes
Release-specific context not captured in changelogs.
*   Format: `vX.Y.Z_notes.md`
